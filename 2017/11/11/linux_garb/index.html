linux环境vi、vim 中文乱码

问题描述：

Linux 服务器vi、vim 乱码，cat、more正常

解决方案：

vi乱码

vi ~/.virc 添加
set fileencodings=utf-8,gb2312,gb18030,gbk,ucs-bom,cp936,latin1
set enc=utf8
set fencs=utf8,gbk,gb2312,gb18030
:wq!保存退出

vim乱码

vi ~/.vimrc 添加
set fileencodings=utf-8,gb2312,gb18030,gbk,ucs-bom,cp936,latin1
set enc=utf8
set fencs=utf8,gbk,gb2312,gb18030
:wq!保存退出
